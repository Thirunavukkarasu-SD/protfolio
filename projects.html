Flagship Project · Academic

Smart Classroom Monitoring System

Analyze student attendance, attention, and classroom activity through computer vision.

Problem statement

Manual classroom monitoring is inconsistent and hard to scale across sessions. Faculty need measurable student engagement indicators without increased administrative overhead.

Solution overview

A local ML pipeline processes classroom video, tracks attendance signals, estimates attention patterns, and provides summarized insight in a Streamlit dashboard.

Architecture

  • Video input and frame processing with OpenCV.
  • Object and person understanding using YOLOv8.
  • Behavior and sequence analysis using CNN + LSTM.
  • Dashboard and reporting interface in Streamlit.

Tech stack

  • Python
  • YOLOv8
  • OpenCV
  • CNN + LSTM
  • Streamlit

Privacy and deployment

Implemented as a local workflow to keep classroom data in controlled environments.

Results and learnings

Working local demo completed with practical end-to-end computer vision integration.

Secondary Project · Personal

RAG-based AI Assistant

Answer questions from private documents with no internet dependency.

Problem statement

Cloud assistants can conflict with privacy and compliance needs when sensitive documents are involved.

Solution overview

A fully offline RAG assistant indexes documents locally and returns grounded responses through local inference.

Architecture

  • Document ingestion and chunking pipeline in Python.
  • Local embedding and retrieval with vector database support.
  • RAG orchestration using LangChain.
  • On-device inference with LM Studio and AnythingLLM integration.

Tech stack

  • Python
  • LangChain
  • Vector DB
  • LM Studio
  • AnythingLLM

Privacy and deployment

Designed for local-only execution so private knowledge remains offline.

Results and learnings

Implemented as a local personal assistant workflow with privacy-first constraints.

Privacy-first systems, built for practical deployment.
